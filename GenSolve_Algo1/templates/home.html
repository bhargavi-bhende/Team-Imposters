About the Algorithm

Welcome to our shape identification and regularization application! This tool utilizes advanced algorithms to analyze and visualize geometric shapes from input data. Below is an overview of what the system does:

1. Shape Identification

The application is designed to identify and regularize several types of geometric shapes:

  • Line: Detected by fitting a linear regression model to the provided data points. The algorithm evaluates the goodness of fit to determine if the shape is a line.
  • Circle: Identified by fitting a circle equation and checking the uniformity of distances from the center. The algorithm calculates the circle's center and radius.
  • Doodle: The system can recognize and regularize freehand drawings into standard geometric shapes like circles or rectangles, ensuring the doodle conforms to the intended shape.
  • Rectangle: Determined by verifying distances between points and checking the equality of diagonals to ensure a rectangular shape.

2. SVG Generation

Once the shapes are identified, they are rendered into SVG format for high-quality vector visualization. The SVG files are then converted into PNG format for easy viewing and sharing.

3. How It Works

The process involves reading input data from a CSV file, identifying shapes using the outlined algorithms, and then generating and saving visual representations of these shapes.
